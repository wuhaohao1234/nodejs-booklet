# 版本管理

语义化版本（semver）即 dependencies、devDependencies 和 peerDependencies 里的如："co": "^4.6.0"。 

1. 主版本号1.x.x
2. 次版本号x.1.x
3. 修订号x.x.1

major: 主版本号

premajor: 预备主版本号

minor: 次版本号

preminor: 次备用版本号

patch: 修订版本号

prerelease: 预发布版本号

npm scripts常用命令

npm update name -S 更新生产环境安装包

npm update name -D 更新开发环境包

npm version 版本号 更新npm版本
